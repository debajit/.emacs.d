** Smart Shortcuts

*** s-. ‚Äî Open the item at Point

In programming modes, this will open the definition/declaration of the
item (the function, for instance) ‚Äî and ~s-.~ will take you back to
where you came from.

If you‚Äôre on a filename or a hyperlink (e.g. in a comment or in a
string), it will open that.

If you‚Äôre in an Org mode link to a URL or local file, it will open
that instead.

*** s-r ‚Äî Run the Code

If some code is selected, that will be run. Otherwise the entire
buffer will be run by calling the appropriate interpreter/tool.

*** s-n ‚Äî Narrow or Widen Intelligently

If you‚Äôre in a programming mode, it will narrow to the function.

In Org mode, narrows to the current sub-tree if any.

If invoked from a selection, it will narrow to that selection.

If you‚Äôre already narrowed, it will widen automatically.
